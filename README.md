# 🚑 Sanjeevni – Emergency Hospital Finder

## Overview

Sanjeevni is a full-stack web application designed to help users identify the most suitable hospitals in real time during emergencies.
It uses a weighted ranking system to prioritize hospitals based on critical factors, enabling faster and more informed decision-making.

---

## 🚀 Features

* Real-time hospital discovery
* Smart weighted ranking algorithm
* Emergency SMS alerts using Twilio API
* One-tap SOS functionality
* Fully responsive user interface

---

## 🧠 Ranking Algorithm

Hospitals are ranked using a weighted scoring system:

* Readiness: 40%
* Beds Availability: 25%
* Distance: 20%
* Ratings: 15%

This approach ensures users are directed to the most relevant hospital instantly.

---

## 🛠 Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB
* APIs: Twilio SMS API

## 🌐 Deployment
